Comandos CMD e PowerShell para Diagnóstico

Como abrir o CMD

✅ Passo 1: Abrir o CMD (Prompt de Comando)
Clique no botão Iniciar (com o símbolo do Windows).
Digite: cmd e selecione "Prompt de Comando".

✅ Passo 2: Digitar o comando
Com o CMD aberto, digite ou cole os comandos abaixo.

⚠️ O uso incorreto pode danificar o sistema.

Comandos para Diagnóstico

wmic cpu get name

Mostra o nome/modelo do processador.

wmic memorychip get DeviceLocator, Capacity, Speed, MemoryType

Exibe slot, capacidade, velocidade e tipo da RAM.

  • DeviceLocator: Slot onde o módulo está instalado.
  • Capacity: Capacidade de cada módulo (em bytes).
  • Speed: Velocidade em MHz.
  • MemoryType: Tipo da memória (DDR2, DDR3, DDR4, etc.). O número retornado precisa ser interpretado com base na tabela abaixo.

🧠 Códigos de MemoryType e seus significados

Código Tipo

wmic diskdrive get model, size

Mostra o modelo e tamanho dos discos.

wmic baseboard get product, Manufacturer

Exibe a placa-mãe.

dxdiag

Abre a ferramenta de diagnóstico DirectX.

driverquery

Lista todos os drivers instalados.

PowerShell: Get-WmiObject MSAcpi_ThermalZoneTemperature

Exibe temperatura da CPU (pode falhar).

ipconfig /all

Exibe todos os detalhes da rede, incluindo MAC, IP, DNS e DHCP.

ping www.google.com

Testa a conexão com a internet e o tempo de resposta.

netstat -an

Exibe todas as conexões de rede ativas e suas portas.

tasklist

Lista todos os processos ativos no sistema (semelhante ao Gerenciador de Tarefas).

systeminfo

Exibe informações detalhadas do sistema, como nome do PC, build do Windows e memória.

cleanmgr

Abre o utilitário de limpeza de disco do Windows.

ipconfig /flushdns

Limpa o cache de DNS (útil para resolver problemas de rede).

del /q /f /s %TEMP%\*

Remove todos os arquivos temporários do sistema.

wsreset

Limpa o cache da Microsoft Store (resolve problemas de instalação).

defrag C: /O

Desfragmenta e otimiza a unidade C: (melhora desempenho em HDD).

DISM /Online /Cleanup-Image /StartComponentCleanup

Limpa atualizações antigas do Windows para liberar espaço.

Gerar Script para Execução






O que faz cada comando:

  • sfc /scannow: Verifica e repara arquivos do sistema corrompidos ou ausentes usando o "Verificador de Arquivos do Sistema".
  • chkdsk /f /r: Verifica e corrige erros no disco rígido. /f corrige erros e /r tenta recuperar dados em setores defeituosos.
  • DISM: Repara a imagem do sistema do Windows usada pelo SFC. Use se o sfc não resolver os problemas.
  • format X:: ⚠️ Apaga todos os dados da unidade X:. Só use se tiver certeza.
